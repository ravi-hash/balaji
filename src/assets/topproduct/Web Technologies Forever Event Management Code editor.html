    Forever Event Management
    Description


    Forever is an Event management company that helps you conduct creative, feasible, efficient and meticulously planned events at affordable prices. They are inviting vendors who would like to collaborate with them. Design a Vendor registration form towards this requirement - adhering to the following specifications.

    Within the form (given in template):
    Figure Element: The id of the figure element must be forever_image. This element contains an image with a text associated with it.
    Image Element: The img component within the figure must be specified with an id: image1 and appropriate src.
    Table Element: Id of the table caption must be: table_caption and the caption text must be: Vendor Registeration Form

    Label

    Element Id

    Constraints

    Company name

    cname

    To enter the company's name.
    Design Constraint: The field is mandatory.

    Type

    Use "corporation", "partnership", "individual" and "others" as the component ids for the radio buttons.

    To select the vendor type.
    Design Constraint: The name of the radio button is 'ctype'.

    Phone Number

    phno

    To enter the phone number.
    Design Constraints: Use the input type specific to the phone number. This mandatory field must accept a 10-digit mobile number that starts with:7,8 or 9.

    Email ID

    email

    To enter the email id.
    Design Constraints: This mandatory field must use the input type specific to the email id.

    Location

    address

    To enter the location.
    Design Constraints: Specify a right tag with the right attributes to create a text area with 4 rows and 50 columns. The element is mandatory.

    Website Address

    link

    To enter the website address.
    Design Constraints: Use the input type specific to the website address. The element is mandatory.

    TIN No

    tin

    To enter the TIN No.
    Design Constraints: Specify the input type such that the field accepts only a number. The element is mandatory.

    TIN No Expiry Date

    expiry

    To enter the TIN No expiry date.
    Design Constraints: Specify the input type such that the field accepts only a date. The element is mandatory.

    Average Project Cost

    cost

    To specify the average project cost.
    Design Constraints: Specify the right input type to create a range slider with min value:25000 and max value:500000. The range has to step up in 1000's. The element is mandatory.

    [Remember to include: onchange="show_value(this.value);" as an attribute-value in this element. This will invoke the javascript given in the template & you will see the difference as you slide through the range slider.]

    Upload Images of Licence, PAN & Facade

    profile

    To upload images of licence, PAN & facade
    Design Constraints: Specify the right input type to make this element accept only files. The element must be able to accept multiple files and is mandatory.


    submit

    Within the div (with id:"foot"), submit the element with type "submit" and value "REGISTER" must be placed. It is clickable but has no action associated with it.


    reset

    Within the div (with id:"foot"), reset the element with type "reset" and value "CLEAR" must be placed. It is clickable but has no action associated with it.
